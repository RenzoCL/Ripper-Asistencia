# =============================================================
# requirements.txt — Servidor (PC central con el backend)
# =============================================================
# Instalar con:
#   pip install -r requirements.txt
#
# En Windows antes de instalar face-recognition necesitas:
#   1. Visual Studio Build Tools 2019+ con "C++ build tools"
#      https://visualstudio.microsoft.com/visual-cpp-build-tools/
#   2. cmake: pip install cmake
#   3. Luego: pip install dlib face-recognition
#
# En Ubuntu/Debian antes de instalar face-recognition:
#   sudo apt-get update
#   sudo apt-get install -y build-essential cmake libopenblas-dev liblapack-dev
#   sudo apt-get install -y libx11-dev libgtk-3-dev python3-dev
#   pip install dlib face-recognition
# =============================================================
